# Day 003 — The Equalizer

## Problem
- Platform: Codeforces
- Link: https://codeforces.com/problemset/problem/2040/A
- Difficulty: 800

## What I Learned
- Game theory problems can be solved using simple parity (odd/even) logic
- If sum of array is odd, first player wins without special move
- If n*k is even, first player wins by using the special move
- BufferedReader is faster than Scanner for Codeforces Java submissions
